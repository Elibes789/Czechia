# High-Speed Bumper

A small high-speed 2D platformer built with HTML5 Canvas and vanilla JavaScript.

Play by opening `index.html` on a static server (for example: `python -m http.server 8000`) and visit http://localhost:8000 in a browser.

Features:
- Endless chunked world generation
- High-speed movement, dash, double-jump
- Throwable boxes that explode
- Enemies that explode and chain-react
- Walls, collapsing platforms, and boost hoops
- Mobile touch controls and audio sample upload support

Files:
- `index.html` — main page and UI
- `style.css` — styles
- `main.js` — game logic
